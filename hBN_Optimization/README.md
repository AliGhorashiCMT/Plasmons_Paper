# Optimization of hexagonal Boron Nitride lattice

To ensure proper convergence, the bash script is run several times. The full output can be found in ./hBN-convergence.out, from the very first lattice iteration to the last. The convergence is started at a lattice constant of 5 Bohr. The algorithm converges to a lattice constant of 4.72749 Bohr, which is within 1 percent of the lattice constant reported in https://materialsproject.org/materials/mp-984/
